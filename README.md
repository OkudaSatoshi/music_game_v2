# MusicGameVer2
  
# Usage for dist
-windiows only-  
Double clikck soundgame.exe  
# Configs for MusicGameVer2
config.jsonの値を変更することで、ゲーム設定を一括で変更することができる  
# Configs for songs
songs.jsonにそれぞれの曲のconfigが書いてあるのでそれを自分で設定する(これはそれぞれEasy、Normal、Hardの難易度で使うmidiファイルを紐づけたり、固有の背景を追加する)
